什么是 Hexo？

Hexo 是一个快速、简洁且高效的静态博客框架。它使用 Node.js 编写，具有以下特点：

  • ⚡ 极速生成：毫秒级生成静态网站
  • 🎨 丰富的插件：支持众多第三方插件
  • 📝 Markdown 支持：使用 Markdown 编写文章
  • 🚀 一键部署：支持 GitHub Pages、Netlify 等多种部署方式
  • 🔧 灵活配置：高度可定制的配置选项

什么是 Fluid 主题？

Fluid 是一个响应式、精美的 Hexo 主题，设计简洁而不失功能性。Fluid 的优势包括：

  • 🎨 现代设计：采用现代化的设计理念
  • 📱 完全响应式：在各种设备上都能完美显示
  • 🌙 深色模式：内置深色模式支持
  • 🔍 本地搜索：支持全文搜索功能
  • 💬 评论系统：集成多个评论系统（Gitalk、Valine 等）

常见问题解决

Q: 主题不生效？

A: 确保主题名称与 _config.yml 中 theme 配置一致，并重启本地服务器。

Q: 图片无法显示？

A: 确保图片路径正确。建议将图片放在 source/img/ 目录下，使用 /img/filename 引用。

Q: 如何添加搜索功能？

A: 需要安装搜索插件，参考 Fluid 官方文档配置本地搜索。

性能优化建议

  1. 启用 gzip 压缩：减小文件传输大小
  2. 使用 CDN：加速资源加载
  3. 图片优化：使用 WebP 或压缩图片
  4. 缓存策略：合理设置浏览器缓存
  5. 代码分割：按需加载 JavaScript
